# CoffeeEnhancer

A gem to make coffeescript even more fun to use.


## Installation


```Node
npm install 'coffee_enhancer_node'

To setup in your browser:

	http://browserify.org/

```

```Meteor
npm install 'coffee_enhancer_node'

To setup in your browser,
follow this tutorial:
	
	https://react-in-meteor.readthedocs.org/en/latest/client-npm/
```
